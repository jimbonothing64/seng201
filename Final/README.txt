To open program source code in eclipse:
    1. Go to file -> open projects from filesystem
    2. Click "Directory" button next to input source and navigate to the same directory
       as this README.txt. Then navigate to Java and click on "Game" to highlight it
    3. Click select folder in the bottom of the window
    4. Click finish.
  There should now be a new project called Game in your eclipse enviroment.

To run the game from source:
  When in eclipse run "Main" class, located in Game/src/Game/Main.java
